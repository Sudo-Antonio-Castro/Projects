Power Supplies and Cooling

A PC depends on a stable power supply and consistent airflow. When either one fails, symptoms range from random shutdowns to permanent component damage.

Power Supply Basics

  • The power supply unit (PSU) converts AC input from the wall into regulated DC output for the motherboard and components.
  • Most desktops use an ATX-style PSU with a 24-pin motherboard connector plus dedicated CPU, SATA, Molex, and PCIe power leads.
  • Always confirm outlet voltage support and connector compatibility before installation.

Wattage, Rails, and Efficiency

  • A PSU must provide enough total wattage for the combined load of the CPU, GPU, drives, fans, and expansion cards.
  • Modern systems rely heavily on the +12V rail, so total wattage alone is not enough - rail distribution matters.
  • 80 PLUS ratings help compare efficiency; a more efficient PSU wastes less power as heat and usually runs cooler and quieter.

Cooling Methods

  • Passive heat sinks spread heat without a fan, while active heat sinks use a fan to improve convection over fins.
  • Thermal paste or a thermal pad improves contact between a chip and its cooler by filling tiny air gaps.
  • Good airflow depends on clean vents, unobstructed fans, and neat cable routing. High-end systems may use liquid cooling or an all-in-one cooler for the CPU.
Summary

Think of the PSU and cooling system as the foundation of reliability. A correctly sized, efficient PSU and a clean airflow path prevent instability long before troubleshooting begins.

A+ Exam Tip

A system that powers on but shuts down under load often points to overheating or an underpowered PSU. For the exam, remember 24-pin ATX main power, CPU supplemental power, and the difference between passive and active cooling.

Storage Devices

Mass storage keeps data when power is off. CompTIA A+ expects you to compare drive technologies, choose the right interface, and understand common redundancy options.

HDDs vs SSDs

  • Hard disk drives (HDDs) use spinning platters and moving heads. They offer large capacity at low cost but are slower and more sensitive to shock.
  • Solid-state drives (SSDs) use flash memory, so they are faster, quieter, and more durable than HDDs.
  • Flash-based storage relies on wear leveling to spread writes across memory cells and extend the usable life of the drive.

Interfaces and Form Factors

  • Common internal form factors include 3.5-inch desktop drives, 2.5-inch laptop drives, and M.2 modules.
  • SATA can be used with HDDs or SSDs, while NVMe over PCIe is the high-performance option for many M.2 SSDs.
  • External storage can use USB, Thunderbolt, or memory cards such as SD and microSD, depending on the device and speed requirement.

RAID and Removable Media

  • RAID 0 stripes data for speed but offers no fault tolerance. RAID 1 mirrors data for redundancy.
  • RAID 5 uses distributed parity and can survive one disk failure, while RAID 10 combines striping and mirroring for strong performance and resilience.
  • Optical discs and flash media are still encountered in support roles, especially for transfer, recovery, or specialty device workflows.
Summary

Choose storage by balancing capacity, speed, endurance, and recovery needs. SSDs dominate performance-sensitive tasks, while RAID is about availability as much as speed.

A+ Exam Tip

For A+, keep the physical form factor separate from the protocol. M.2 describes the card shape; SATA or NVMe describes how the device communicates.

System Memory

RAM provides the fast working space the CPU needs for active programs and data. Memory upgrades are one of the most common and cost-effective performance improvements.

Memory Fundamentals

  • RAM is volatile storage, which means it loses contents when power is removed.
  • Desktop systems usually use DIMMs, while laptops commonly use SODIMMs.
  • DDR generations are keyed differently and are not interchangeable across slots.

Characteristics and Types

  • Key memory characteristics include capacity, transfer rate, channel configuration, and latency.
  • DDR5 improves base speed and uses dual 32-bit subchannels per DIMM, while older generations used a single 64-bit channel.
  • ECC memory adds error checking and is common in servers and workstations; most standard desktops use non-ECC RAM.

Installation and Troubleshooting

  • Follow motherboard documentation for the correct slot population order when using dual-channel or higher channel modes.
  • Handle memory by the edges and fully seat it until the locking tabs click into place.
  • Bad or mismatched RAM can cause POST failures, random reboots, blue screens, and application crashes.
Summary

System memory affects responsiveness, multitasking, and stability. More RAM helps only when the platform supports it and the modules are installed in the correct slots and type.

A+ Exam Tip

A+ often tests three ideas together: memory generation compatibility, slot population order, and the symptoms of faulty or improperly seated RAM.

CPUs

The CPU is the main processing component in a computer. To support or upgrade one, you need to understand compatibility, performance features, and safe installation practice.

Performance Factors

  • CPU performance is shaped by clock speed, core count, thread count, cache size, and architecture.
  • Modern processors may also support virtualization extensions, integrated graphics, power-management features, and security instructions.
  • Thermal design power (TDP) helps estimate cooling requirements and expected heat output.

Compatibility and Socketing

  • A processor must match the motherboard socket and chipset. Intel and AMD use different socket families.
  • Desktop CPUs may be replaceable, but many mobile CPUs are soldered to the motherboard and are not practical to upgrade.
  • Always verify firmware support before installing a newer processor into an older board.

Installation Basics

  • Align the processor correctly using the orientation marker, lock it into the socket, then apply thermal paste if required.
  • Install the heat sink and fan assembly firmly and connect the cooler to the proper CPU fan header.
  • Too much paste, too little paste, or a loose cooler can all lead to overheating and throttling.
Summary

CPU upgrades are never just about the chip. The motherboard, firmware, cooler, and power delivery all have to support the processor you choose.

A+ Exam Tip

For the exam, remember the sequence: confirm compatibility, seat the CPU correctly, apply thermal interface material, secure the cooler, and connect the CPU fan.
