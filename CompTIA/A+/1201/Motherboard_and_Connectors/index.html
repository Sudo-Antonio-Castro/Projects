Personal Computers & Case Anatomy

PC components are divided into two categories: user-handled peripheral devices and internal components housed inside the system case. Understanding this distinction is the foundation for any hardware support role.

Peripheral Devices

  • Input devices: keyboard, mouse, microphone, camera
  • Output devices: monitor/display, speakers
  • External storage: USB drives, external hard drives

Internal Components (inside the case)

  • Motherboard, CPU, RAM modules, adapter cards, fixed disks, power supply unit (PSU)
  • All-in-one PCs contain internal components inside a case that is also the monitor

Tower Case Panels

  • Side cover: slides off, held by screws or clips; may have anti-tamper mechanisms — always follow documentation
  • Front panel: access to removable media drives, power switch, and LEDs; may require removing the side panel first to access its screws
  • Rear panel: PSU sockets (includes integral fan exhaust — keep unobstructed for cooling), motherboard I/O cutout, and expansion card slots

The rear I/O area uses an I/O shield to block dust. Expansion card slots use either a card or a blanking plate to prevent gaps that allow dust entry, electrostatic discharge (ESD), and electromagnetic interference (EMI).

A+ Exam Tip

Know the difference between peripheral devices and internal components. The I/O shield and blanking plates are not cosmetic — gaps in the rear panel increase ESD and EMI exposure. The exam may ask why blank slots must always be covered.

USB Cables & Connector Types

USB (Universal Serial Bus) is the standard for connecting most peripheral devices to a computer. A single USB host controller can support up to 127 devices. Most PC motherboards include multiple controllers, each with 3–4 ports.

USB 2.0 Connector Form Factors

  • Type A: flat rectangular connector for host and some peripheral devices; USB symbol faces up on insertion
  • Type B: square with a beveled top; used on large devices like printers
  • Type B Mini: smaller connector for older peripherals like early digital cameras; now rarely used
  • Type B Micro: flatter connector for smartphones and tablets; very common on USB 2.0 devices

USB 3.x / USB4 Connectors

  • Type A (3.x): same shape as USB 2.0 Type A but with extra signaling pins; often distinguished by a blue connector tab or housing — physically compatible with USB 2.0 ports
  • Type B (3.x): physically larger than USB 2.0 Type B; not backward compatible with USB 2.0 Type B ports
  • Micro Type B (3.x): dual-layer design with extra contacts; not compatible with USB 2.0 Micro ports
  • Type C: compact, reversible, durable design introduced with USB 3.1; can be used at both cable ends; supports USB4 and Thunderbolt 3/4/5

Cable Length & Power

  • Max cable length: 3 m (LowSpeed), 5 m (FullSpeed/HighSpeed), ~3 m recommended for SuperSpeed
  • Basic USB ports supply up to 4.5 W (depending on version)
  • USB Power Delivery (PD) ports supply up to 100 W with compatible connectors and cables
A+ Exam Tip

USB 3.0 Type A is backward compatible with USB 2.0 Type A ports — but USB 3.x Type B and Micro Type B are NOT compatible with their USB 2.0 counterparts. The blue color coding on USB 3.x ports and connectors is a key visual indicator the exam tests on.

Display Technologies

Liquid Crystal Displays (LCDs) are the most common display type, using a backlight and liquid crystals to produce an image. There are three main LCD subtypes, plus newer OLED and Mini-LED technologies.

LCD Subtypes

  • In-Plane Switching (IPS): liquid crystals align parallel to the screen for consistent light transmission and color reproduction
    • ✓ Superior color accuracy and wide viewing angles — ideal for graphic design and professional use
    • ✗ Slower response times than TN, which can affect fast-paced gaming
  • Twisted Nematic (TN): liquid crystals twist 90° between electrodes to control light passage
    • ✓ Faster response times and higher refresh rates — preferred for gaming
    • ✗ Poorer color reproduction and limited viewing angles
  • Vertical Alignment (VA): liquid crystals align vertically to the glass and tilt when voltage is applied
    • ✓ Better color accuracy and viewing angles than TN; good contrast ratios
    • ✗ Slower response than TN; can exhibit color shifting at extreme angles

OLED & Mini-LED

  • Organic Light-Emitting Diode (OLED): each pixel emits its own light — no backlight needed
    • ✓ Excellent color accuracy, contrast ratios, energy efficiency, and true blacks
    • ✗ Potential for burn-in when static images are displayed for long periods
  • Mini-LED: an LCD enhancement using thousands of tiny LEDs for more precise backlight control
    • ✓ Improved brightness, better black levels, reduced halo effect; thin and lightweight designs
    • ✗ More expensive; does not achieve true blacks of OLED; more complex to manufacture
A+ Exam Tip

Mini-LED is an enhancement for LCDs — not a type of OLED. Know the key distinction: OLED pixels are self-emissive (each pixel is its own light source), while LCD panels (including Mini-LED) still use a backlight. The exam may present a scenario and ask which display type best suits it — match color accuracy to IPS/OLED, gaming speed to TN.

Display Components & Attributes

Understanding display components and attributes is essential for evaluating the performance and suitability of various screens for specific use cases.

Touch Screens & Digitizers

  • Integral for interactive displays — allow direct interaction through touch or stylus
  • Common in smartphones, tablets, and some laptops
  • A digitizer is the component that translates physical touch into digital input

Inverters (Legacy LCDs)

  • Older LCDs required an inverter to convert DC power to AC for the fluorescent backlight
  • Modern displays with LED backlighting do not require an inverter
  • A failing inverter in older LCDs often causes the screen to appear very dim or flicker

Key Display Attributes

  • Pixel Density: higher density (more pixels per inch) results in sharper images
  • Refresh Rate: measured in Hz; higher rates provide smoother motion — critical for gaming and video
  • Screen Resolution: Full HD (1920×1080), Quad HD (2560×1440), 4K (3840×2160), 5K (5120×2880), 8K (7680×4320)
  • Color Gamut: the range of colors a display can produce based on the RGB model; common standards are sRGB, Adobe RGB, and DCI-P3 (DCI-P3 offers the widest range)
  • Color Depth: 24-bit color (true color) supports ~16.7 million colors; 32-bit adds an alpha channel for transparency and smoother gradients

Video Cable Bandwidth

Video cable bandwidth is determined by resolution (measured in pixels) and refresh rate (measured in Hz or fps). For example: uncompressed HD at 60 fps requires 4.5 Gbps; 4K at 60 fps requires 8.91 Gbps.

A+ Exam Tip

Frame rate (fps) describes the video source; hertz (Hz) indicates the display's refresh rate. To avoid ghosting and tearing, the refresh rate should match or be divisible by the frame rate — for example, a 60 fps video plays smoothly on a 120 Hz display. A dim or flickering screen on an older LCD typically points to a failed inverter.

HDMI & DisplayPort Video Cables

Both HDMI and DisplayPort are standard digital video interfaces for connecting displays to computers and devices. Choosing the right one depends on use case and device support.

HDMI (High-Definition Multimedia Interface)

  • Functionality: transmits both high-definition video and audio signals through a single cable
  • Standard use: televisions, monitors, gaming consoles, projectors — anywhere consumer electronics need a simple connection
  • Versions: newer versions support higher resolutions, increased refresh rates, and HDR (High Dynamic Range) for enhanced color and contrast
  • Advantages: single-cable solution; broad compatibility with a wide range of devices; widely adopted in consumer electronics
  • Note: HDMI supports only video and audio — not data transfer or power delivery

DisplayPort

  • Purpose: digital display interface primarily for connecting a video source (computer) to a display device (monitor)
  • Preferred use: professional environments, gaming setups, multi-monitor configurations
  • Key features: supports higher resolutions and refresh rates than HDMI; Multi-Stream Transport (MST) allows multiple displays to connect through a single port
  • Additional capabilities: transmits audio; supports adaptive sync technologies (AMD FreeSync and NVIDIA G-Sync) to reduce screen tearing

A+ Exam Tip

DisplayPort's MST (Multi-Stream Transport) is a key differentiator — it lets you daisy-chain multiple monitors from a single port. HDMI is the standard for consumer electronics and carries audio. Neither DVI nor VGA carries audio. The exam may ask you to select the correct cable for a given scenario.

Originally associated with Apple, Thunderbolt is now widely implemented on Windows and Linux PCs as well. It can function as both a display interface (like DisplayPort) and a general peripheral interface (like USB).

Lightning Interface (Apple)

  • Proprietary Apple connector used primarily in iPhone and iPad devices introduced in 2012
  • Replaced the older 30-pin dock connector with a more compact, reversible design
  • Requires a Lightning-to-USB-A or Lightning-to-USB-C adapter cable to connect to PCs
  • Being phased out in favor of USB-C, driven by industry standardization (USB-C is more versatile, faster, and higher power delivery), regulatory pressure (EU common charging standard), and technological advancement
A+ Exam Tip

Thunderbolt 3 and later use the USB-C physical connector — but a USB-C port is NOT the same as a Thunderbolt port. TB3 devices won't work in non-Thunderbolt USB-C ports. The lightning bolt icon on the port is how you identify Thunderbolt. The exam may also ask about Lightning adapters for connecting Apple devices to PCs for charging and data transfer.

SATA & Molex Power Connectors

Serial Advanced Technology Attachment (SATA) is the standard interface for connecting internal storage drives (HDDs and SSDs) in desktop PCs. Separate data and power cables are always required.

SATA Data Interface

  • Uses compact 7-pin data connectors with cables up to 1 meter in length
  • Each cable supports a single device — unlike older IDE/PATA which supported two devices per cable
  • Speed revisions: SATA 1 = 150 MB/s, SATA 2 = 300 MB/s, SATA 3 = 600 MB/s
  • SATA 3.1–3.5 added features (SATA Express, Universal Storage Module) but did not increase speeds

SATA Power Connector

  • Separate 15-pin power connector — the 7-pin data connector does not carry power
  • Newer SATA power connectors include an orange 3.3V wire in addition to the standard 5V and 12V rails

Molex Power Connectors (Legacy)

  • 4-pin Molex KK connector used on older/legacy storage components and some fans
  • Color codes: red = 5V DC, yellow = 12V DC, black = ground
  • Made of white or clear plastic; some devices have both SATA and Molex power connectors

External SATA (eSATA)

  • Allows peripheral drives to connect via cable up to 2 meters (78 inches)
  • eSATA cables are required — internal SATA cables are not compatible with eSATA ports
  • eSATAp (also called "eSATA/USB Combo") adds USB functionality to the same port
  • Less common today — USB 3.x and Thunderbolt offer higher speeds, power delivery, and broader adoption
A+ Exam Tip

SATA 3 = 600 MB/s maximum. Know that the 7-pin connector is data-only and the 15-pin connector provides power — you always need both. Molex wire color codes are testable: red=5V, yellow=12V, black=ground. eSATA does not supply power over the cable — a limitation that USB 3.x and Thunderbolt do not have.

Motherboard Functions & Overview

All computer software and data are processed using binary code (ones and zeros). The central processing unit (CPU) executes software instructions — this is the "compute" function of a PC. Because the CPU can only store a limited number of instructions internally, it relies on additional resources.

  • System Memory (RAM): provides working storage for programs and open data files; RAM is nonpersistent — it only holds data while the PC is powered on
  • Mass Storage: HDDs and SSDs preserve data when the computer is turned off
  • Bus interfaces on the motherboard connect all processing and storage components via electrical signals
  • The system clock synchronizes all PC operations, measured in megahertz (MHz) or gigahertz (GHz)
  • Clock multipliers adjust the timing signal to produce different speeds for various buses

Major Manufacturers

  • CPU manufacturers: Intel and Advanced Micro Devices (AMD)
  • Motherboard manufacturers: AOpen (Acer), ASRock, ASUSTek, Biostar, EVGA, Gigabyte, Intel, MSI
  • Each motherboard supports specific CPU ranges — a board for Intel CPUs will not accept AMD CPUs and vice versa
Summary

The motherboard is the backbone of a PC — it provides the physical connections and bus interfaces that allow the CPU, RAM, storage, and expansion cards to communicate. Understanding the memory hierarchy (CPU cache → RAM → mass storage) helps explain why performance degrades when systems run low on RAM or use slow storage. The chipset soldered onto the motherboard determines which CPUs, how much RAM, and which interfaces are supported — it cannot be upgraded.

Electrical Safety & Electrostatic Discharge (ESD)

Electrical Safety

  • Always disconnect the PC from the power supply before opening the case
  • After unplugging, hold the power button for a few seconds to drain any remaining charge from internal components
  • Never attempt to disassemble non-field-repairable components such as the power supply unit

Electrostatic Discharge (ESD)

ESD occurs when a static charge from your clothes or body is suddenly released into a circuit by touch. Components most vulnerable to ESD: CPU, system RAM, adapter cards, and the motherboard itself.

  • Handle components by their edges or plastic parts — never touch circuit traces or connector pins
  • Use an anti-ESD wrist strap connected to a grounded surface or ESD mat whenever working inside a PC
  • Store components in anti-static bags when not installed
  • The I/O shield's pins connect external metal parts to the metal case and PSU, providing a safe path for ESD to ground
A+ Exam Tip

ESD can permanently damage sensitive components with no visible signs of damage — the system may fail intermittently or stop working entirely days later. The anti-ESD wrist strap is the primary protective equipment. Never work on components placed on carpet or other insulating surfaces. Operational safety procedures are covered in more detail in Core 2.

CPU Sockets & the Chipset

CPU Socket

  • The CPU socket has a distinctive square shape and is typically located near the center-top of the motherboard
  • Intel and AMD use different socket designs — a CPU from one manufacturer will not fit in the other's socket
  • Due to rapid advancements in CPU technology, a motherboard supports only a limited range of processor models
  • After installing the CPU: apply thermal paste, then attach the heat sink and fan to manage heat

The Chipset

  • The chipset is soldered onto the motherboard and cannot be upgraded
  • It manages data transfer between the CPU and various devices (RAM, storage, I/O)
  • The chipset determines:
    • Compatible processor models
    • The type and maximum amount of RAM supported
    • Integrated interfaces (video, sound, networking)
  • Interfaces not supported by the chipset can be added or upgraded via adapter/expansion cards
Summary

New motherboards are typically released to support the latest CPU models. Because the socket and chipset are physically fixed, upgrading a CPU sometimes requires a new motherboard. The chipset is the brain of the motherboard — it determines the entire feature set available to that system. When selecting components for a build, always verify chipset compatibility with the chosen CPU and desired features.

A+ Exam Tip

The CPU goes in the socket; the chipset is soldered on and cannot be replaced. Intel and AMD sockets are incompatible — this is a common source of build errors. When installing a CPU: socket → thermal paste → heat sink → fan. Missing thermal paste causes CPU overheating and thermal throttling or shutdown.

System Memory Slots & RAM

System memory uses random-access memory (RAM) technology — volatile storage that loses its contents when power is removed. Program code and data are loaded into RAM for access and execution by the processor.

DIMM Slots

  • System RAM is packaged as Dual Inline Memory Modules (DIMMs) that fit into slots near the CPU socket
  • DIMM slots have catches (clips) at either end, and are often numbered and color-coded for dual-channel configuration guidance
  • Labels next to the slots indicate the type of DIMMs supported
  • The memory controller's capabilities and the number of physical slots determine the maximum amount of memory that can be installed

DDR Generations

GenerationKey FeatureChannel Width
DDR3Single 64-bit channel per DIMM; motherboard-controlled voltage64-bit
DDR4Higher base speeds; improved efficiency; still 64-bit per DIMM64-bit
DDR5Dual 32-bit subchannels per DIMM; On-DIMM PMIC (power management IC); significantly higher base speeds2×32-bit

Each DDR generation uses a different physical DIMM form factor and is not interchangeable — DDR4 RAM will not fit in a DDR3 slot and vice versa.

A+ Exam Tip

DDR generations are not backward compatible. The key DDR5 advancement is the shift from a single 64-bit channel to dual 32-bit subchannels per DIMM, plus on-DIMM power management. When installing RAM, follow the motherboard's documentation for which slots to use for dual-channel configuration — typically slots 1 and 3 or 2 and 4 (color-coded pairs).

Motherboard Storage Interfaces

Fixed disks inside the PC case use either solid-state drive (SSD) or hard disk drive (HDD) technology. The motherboard provides several interfaces to connect them.

SATA Ports

  • Motherboards include several SATA ports to connect fixed drives, removable drives, tape drives, and optical drives (DVD/Blu-ray)
  • SATA devices sit in a drive bay in the chassis, connected via a data cable to the SATA port and a separate power cable (SATA power or Molex) to the PSU

M.2 Interface

  • An SSD can be provisioned in an adapter card form factor using an M.2 interface
  • The M.2 port is oriented horizontally — the adapter card inserts at an angle, is pushed into place, and secured with a screw
  • M.2 adapters come in different lengths: 42 mm, 60 mm, 80 mm, or 110 mm — check your motherboard for compatibility; labels indicate the supported sizes
  • M.2 supplies power over the bus — no separate power cable is needed
  • M.2 slots can support either SATA or NVMe (PCIe) protocol — check which protocols your motherboard's M.2 slot supports
Summary

SATA is the standard for traditional 2.5-inch and 3.5-inch drives and optical drives — it requires separate data and power cables. M.2 is the compact, cable-free form factor for modern SSDs, offering significantly higher speeds when using the NVMe protocol over PCIe. When installing an M.2 SSD, always verify the module length and protocol (SATA vs NVMe) against the motherboard specifications before purchasing.

A+ Exam Tip

M.2 is the physical form factor — not a speed standard by itself. An M.2 SSD using NVMe (PCIe) is much faster than one using SATA. The exam may present a scenario asking why an M.2 drive is slow — the answer is likely that it's using SATA protocol rather than NVMe.

PCIe & PCI Expansion Slots

Expansion slots accept plug-in adapter cards to extend the computer's functionality. There are two main types: the modern standard (PCIe) and the legacy standard (PCI).

PCI Express (PCIe)

  • PCIe is the standard interface for modern adapter cards, using point-to-point serial communication to give each component a dedicated link
  • Each connection is called a link, which uses one or more lanes
  • Transfer rates are measured in GT/s (giga transfers per second); throughput in GB/s is the effective rate after encoding losses

  • Slot sizes: x1, x4, x8, x16 — physically larger slots have more lanes
  • Up-plugging: a shorter card in a longer slot (e.g., x8 card in an x16 slot) — will work at the card's native lane count
  • Down-plugging: a longer card in a shorter slot — possible if the card is not physically obstructed
  • All PCIe versions are backward compatible — a PCIe 2 card works in a PCIe 4 slot (at PCIe 2 speeds)
  • Power: PCIe supplies up to 75W via a dedicated graphics slot and up to 25W through other slots; additional 75W can be supplied via a PCIe power connector

Legacy PCI

  • PCI is a legacy parallel bus, superseded by PCIe; PCIe is software-compatible with PCI — legacy PCI adapter cards work in PCIe motherboards via a compatibility mode, but PCI cards cannot physically fit in PCIe slots
  • PCI: 32-bit, 33.3 MHz, up to 133 MB/s transfer rate; keyed for 5V, 3.3V, or dual-voltage cards
A+ Exam Tip

PCIe slots sharing lanes with other components (another PCIe slot or an M.2/NVMe slot) can reduce each other's bandwidth. A slot may be labeled x16 but only operate at x8 if it shares lanes — check the motherboard manual. The exam tests up-plugging vs down-plugging behavior and PCIe backward compatibility.

No commercial motherboards were ever produced from the original plain ITX specification — only ATX, mATX, and Mini-ITX matter for the exam.

Motherboard Installation

Installing a motherboard involves securing it inside the case and connecting various components. Follow these steps in order for a successful installation:

1. Review Documentation 2. Install I/O Shield 3. Insert Standoffs 4. Pre-install CPU & Memory 5. Align & Secure Board 6. Final Assembly 7. Cable Management
  • 1. Review Documentation: familiarize yourself with the installation procedure; check if jumper settings need adjustment (modern boards often use BIOS/UEFI instead of physical jumpers)
  • 2. Install I/O Shield: if the board doesn't come with a pre-installed I/O shield, align it with the rear I/O ports and snap it into the case — do this before the board goes in
  • 3. Insert Standoffs: place standoffs in the case only where the motherboard has corresponding mounting holes — standoffs in the wrong position cause short circuits
  • 4. Pre-install CPU and Memory (Optional but recommended): install CPU, memory, and CPU cooler on the board before placing it in the case — provides easier access and reduces the risk of damage
  • 5. Align and Secure the Motherboard: place the board onto the standoffs aligning with the I/O shield cutout; secure with the appropriate screws without overtightening
  • 6. Final Assembly: connect power supplies, disk drives, and adapter cards; attach all necessary data and power connectors
  • 7. Cable Management: plan and route cables neatly to maintain airflow and prevent clutter inside the case
A+ Exam Tip

Use an anti-static wrist strap throughout the installation. I/O shield must go in before the motherboard — it cannot be installed after. Standoffs that don't align with motherboard holes will cause a short circuit when the case is powered on. Cable management is not cosmetic — poor airflow causes thermal issues and premature component failure.

Motherboard Headers & Power Connectors

In addition to expansion slots and storage connectors, motherboards include connectors for case components and the power supply. Document all header positions before disassembling a system.

Front Panel Headers

  • Power button (soft power): sends a signal the OS interprets as a shutdown command; holding the power button for several seconds bypasses the OS and cuts power directly
  • Drive (HDD) activity lights: LED indicator that shows when an internal disk is being accessed; connector usually labeled "HDD LED" on the motherboard
  • Audio ports: allow speakers, headphones, and microphones to connect; front panel audio typically connects via an HD Audio header (labeled "HD Audio" or "AC'97" on older systems)
  • USB ports: internal USB 2.0 connections use 9-pin headers (supports up to two 4-pin USB ports; the 9th pin ensures correct cable orientation); USB 3.x uses 20-pin (2×10) headers for two USB 3.x ports

Power Connectors

  • Main Power (P1): the primary motherboard power connector is a 24-pin block (2 rows of 12 pins) with square pin receptacles
  • Fan Connectors — 3-pin Molex KK: used for fans; controls speed by varying voltage; 3-pin fans on 4-pin headers work but may lack variable speed control without special configuration
  • Fan Connectors — 4-pin Molex KK (PWM): supports precise fan speed control via pulse width modulation (PWM); there will be one connector for the CPU fan and one or more for case fans
A+ Exam Tip

4-pin fans on 3-pin headers will work but cannot use PWM speed control. 3-pin fans on 4-pin headers work but the system may not be able to vary their speed. USB 2.0 uses a 9-pin header; USB 3.x uses a 20-pin header — they are not interchangeable. When reassembling, refer to the motherboard manual or labels on the wire connectors — header labels are often small and difficult to read.

Expansion Cards

An expansion card enhances a motherboard by adding functions or ports not originally supported. Cards fit into PCIe or PCI slots. Common types include video, capture, sound, and network cards.

Video (Graphics) Cards

  • The video card (graphics adapter) generates the signal for a monitor or projector
  • Low-end graphics are often integrated (onboard) in the motherboard chipset or CPU — sufficient for general use
  • For 3D gaming, CAD, or digital artwork, a discrete graphics card in a PCIe x16 slot is required
  • GPU (Graphics Processing Unit): microprocessor optimized for rendering 2D and 3D images; key metrics are frame rate and texture support
  • Graphics memory (GDDR): high-end cards have up to 24 GB; mid-range typically 8–12 GB; low-end may use shared system RAM
  • Video ports: modern cards feature HDMI, DisplayPort, and sometimes USB-C with Thunderbolt capabilities
  • Most graphics chipsets come from AMD, NVIDIA, and Intel

Capture Cards

  • A capture card records video input and saves it as a file or streams it live — the opposite of a graphics card
  • Game capture cards: record/stream gameplay via HDMI from PC or console
  • HDMI capture cards: record from camcorders, security cameras, and other HDMI sources
  • TV tuner cards: receive and record broadcast TV signals
  • Internal capture cards (PCIe) offer lower latency; external capture cards (USB/Thunderbolt) are portable and easier to install

Sound Cards

  • Speakers and headphones connect via a 3.5 mm audio jack (also called a phone plug or mini TRS connector)
  • A basic sound chip is integrated in most motherboard chipsets; better-quality audio requires a PCIe/PCI expansion card
  • Pro-level sound cards add onboard memory, flash memory for sound samples (wavetables), and additional jack types for different input sources
  • Multi-channel output supports stereo through advanced surround sound systems

Network Interface Cards (NICs)

  • An Ethernet NIC is integrated into most modern motherboard chipsets
  • Add-on NICs are used to upgrade to a different network type (copper vs fiber), add multiple ports for link aggregation, or add wireless capability
  • Wi-Fi adapters support various 802.11 standards; some also support cellular data networks
A+ Exam Tip

GDDR memory in a GPU is different from DDR system RAM — it's optimized for the high bandwidth needs of graphics processing, not general-purpose computing. Integrated graphics share system RAM. A discrete GPU uses dedicated GDDR memory on the card. Capture cards and graphics cards use the same PCIe slots — differentiate them by their function: graphics cards output video, capture cards input video.

Video Graphics Array (VGA)

  • The 15-pin VGA port was the standard analog video interface for PCs for many years
  • Being phased out — most new video cards and monitors no longer include VGA
  • Supports resolutions up to Full HD (1920×1080) depending on cable quality
  • Connector is a D-shell type with screws for secure attachment to the port
  • Originally used for CRT monitors and projectors that rely on analog signals
A+ Exam Tip

DVI-I supports both analog and digital; DVI-D is digital only; DVI-A is analog only. "I" = Integrated (both), "D" = Digital only, "A" = Analog only. VGA is analog only and uses a 15-pin D-shell connector with screws. Neither DVI nor VGA carries audio — always use a separate audio cable or choose HDMI/DisplayPort for combined audio/video.

Serial Cables (RS-232) & Legacy Ports

Serial / RS-232

  • The serial port is a legacy connection interface that transmits data one bit at a time over a single wire
  • Also known as Recommended Standard #232 (RS-232)
  • Supports data rates up to about 115 Kbps — far slower than USB
  • RS-232 specifies a 25-pin interface, but PC manufacturers typically use the cheaper 9-pin D-subminiature (DB9) female port
  • In Windows, the serial port is referred to as a Communications (COM) port

Where Serial Ports Are Still Found

  • Network equipment console ports (routers, switches, firewalls) for device management
  • Legacy industrial control systems and serial printers
  • External modems for dial-up Internet (largely replaced by USB)

PS/2 Ports (Legacy)

  • PS/2 ports use a 6-pin mini-DIN format
  • Color coded: green for mice, purple for keyboards
  • Completely replaced by USB in modern systems but may still appear in legacy environments
A+ Exam Tip

RS-232/serial ports = COM ports in Windows = DB9 9-pin connector. The key use case today is connecting an RS-232 serial cable to the console port on network equipment for device management — this is a common real-world scenario the exam tests. PS/2: green = mouse, purple = keyboard — the color coding is frequently tested.

Adapter Cables

A basic peripheral cable often won't directly connect a PC port to a peripheral device port. An adapter cable solves this by having connectors for two different cable types at each end.

Active vs Passive Adapters

  • Active adapters use circuitry to convert signals (e.g., digital HDMI to analog VGA) — active conversion is required when the two connectors use different signaling standards
  • Passive adapters simply convert between connector types without changing the signal (e.g., USB Type-A to USB Type-C) — only work when both ends use the same signaling standard

USB Hubs

  • USB hubs provide additional USB ports, allowing multiple devices to connect to a single USB port on the PC
  • Powered hubs include their own power supply — useful when connecting multiple high-power devices
  • Unpowered hubs rely on the host port's power budget, which can cause issues with power-hungry devices
A+ Exam Tip

HDMI to VGA always requires an active adapter because it converts a digital signal to analog — passive cables won't work here. HDMI to DVI can be passive because both are digital video. When a USB hub causes devices to malfunction or not be recognized, the fix is usually switching to a powered hub. The exam frequently tests active vs passive adapter scenarios.
