    [KB] Update Winget, Install PowerShell 7, Python, and Ollama on Windows

    Audience: Help Desk / IT Admin System: Windows / PowerShell Last reviewed: 2026-04-06

    This guide walks through updating WinGet sources, installing PowerShell 7, installing Python, installing Ollama, and running the wizard-vicuna-uncensored model on Windows. It provides a simple, repeatable process for preparing a local AI workstation using standard install methods and command-line actions.

    Good for
    Preparing a Windows workstation for local AI usage, standardizing PowerShell and Python installation steps, and getting Ollama ready to run a local language model from PowerShell.
    Be careful
    Run these steps from an elevated PowerShell session when required, verify you have enough disk space and network access for model downloads, and make sure community model usage aligns with your organization's software and AI policy.

    Purpose

    To provide a repeatable, step-by-step process for preparing a Windows system with updated WinGet sources, PowerShell 7, Python, and Ollama, then running a local AI model from PowerShell. This article is intended to standardize the setup flow and reduce install issues during initial workstation preparation.

    When to use this guide

    • When setting up a new Windows workstation for local AI tooling.
    • When PowerShell 7 needs to be installed before additional tooling is used.
    • When Python needs to be installed either from a direct download or by using WinGet.
    • When Ollama needs to be installed and a model needs to be downloaded and run locally.
    Important note

    Some downloads may take time depending on network speed, and the Ollama model download can require significant disk space.

    Procedure

    1. Step 1 — Action: Before using WinGet to install software, update the WinGet sources.
      2. winget source update
    2. Step 2 — Action: Install PowerShell 7.
      3. winget install Microsoft.PowerShell
    3. Step 3 — Action: Install Python on the C:/ drive by downloading it directly.

      4. Primary download:

      https://www.python.org/downloads/

      Alternative download:

      https://www.python.org/downloads/release/pymanager-261/

      Alternative WinGet install:

      winget install 9NQ7512CXL7T
    4. Step 4 — Action: Download and install Ollama.

      5. Primary download:

      https://ollama.com/download/windows

      Alternative PowerShell install:

      irm https://ollama.com/install.ps1 | iex
    5. Step 5 — Action: Install and automatically run the model.
      6. ollama run wizard-vicuna-uncensored
    6. Step 6 — Info: The command above downloads the model if it is not already present, then starts it automatically.
    7. Step 7 — Info: After setup is complete, open PowerShell any time and run the same command again to start the AI model.
      8. ollama run wizard-vicuna-uncensored

    Validation

    • Confirm winget source update completes without errors.
    • Confirm PowerShell 7 launches successfully after installation.
    • Confirm Python is installed in the intended location or is otherwise available for use.
    • Confirm Ollama installs successfully and the model launches from PowerShell.

    Notes

    The first model run may take longer than later runs because it downloads the model locally. Future runs typically start faster as long as the model remains installed on the system.
